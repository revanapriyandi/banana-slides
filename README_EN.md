We long to quickly create presentations that are both professional and well-designed. While traditional AI PPT generation apps generally meet the demand for "speed," several issues persist:

- 1ï¸âƒ£ Only pre-set templates can be selected, with no flexibility to adjust styles.

## ğŸ¯ Features

### 1. Flexible and Diverse Creative Paths

Supports three starting modes: **Idea**, **Outline**, and **Page Description**, catering to different creative habits.
- **One-sentence Generation**: Enter a topic, and the AI automatically generates a well-structured outline and page-by-page content descriptions.
- **Natural Language Editing**: Supports modifying the outline or descriptions via natural language using Vibe (e.g., "Change page three to a case study"), with the AI responding and adjusting in real-time.
- **Outline/Description Mode**: Supports both one-click batch generation and manual adjustment of details.

### 2. Powerful Asset Parsing Capability

- **Multi-format Support**: Upload PDF, Docx, MD, Txt, and other files for automatic background content parsing.
- **Intelligent Extraction**: Automatically identifies key points, image links, and chart information within the text to provide rich materials for generation.
- **Style Reference**: Supports uploading reference images or templates to customize the PPT style.

- **Full-page Optimization**: Generate high-definition, style-consistent pages based on nano banana proğŸŒ.

**FAQ**
1.  **Is the free tier Gemini API Key supported?**
    *   The free tier only supports text generation; image generation is not supported.
2.  **Encountering 503 error or Retry Error during content generation**
    *   You can check the internal Docker logs using the command provided in the README to locate the detailed error for the 503 issue. This is usually caused by incorrect model configuration.
3.  **Why is the API Key not working after setting it in .env?**
    1.  Editing `.env` at runtime requires restarting the Docker container to apply changes.
    2.  If settings were configured via the web settings page, they will override the parameters in `.env`. You can revert to `.env` settings via "Restore Default Settings."
4.  **Garbled characters appearing in the generated page text**
    *   Try using a higher resolution output (the OpenAI format may not support increasing resolution).
    *   Ensure that the specific text content to be rendered is included in the page description.
